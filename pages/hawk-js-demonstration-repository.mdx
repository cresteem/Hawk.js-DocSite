# Demonstration
You can find fabricated Demo and configuration files for **demonstration** in this [GitHub repository](https://github.com/cresteem/Hawk-JS-Demo). Clone this repository to your local machine to **experiment** with configurations to understand how Hawk JS works. This hands-on approach will help you better grasp the functionalities and integration of **Hawk JS**. 

## Clone demonstration repository

```bash filename="Terminal @Demo" copy
git clone https://github.com/cresteem/Hawk JS-Demo.git
```
